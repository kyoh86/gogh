## gogh hook add

Add a new hook

```
gogh hook add [flags]
```

### Options

```
  -h, --help                   help for add
      --name string            Name of the hook
      --repo-pattern string    Repository pattern
      --trigger-event string   event that triggers the hook; it can accept "", "post-clone", "post-fork" or "post-create"
```

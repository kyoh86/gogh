## gogh hook add

Add a new hook

```
gogh hook add [flags]
```

### Options

```
  -h, --help                   help for add
      --name string            Name of the hook
      --repo-pattern string    Repository pattern
      --trigger-event string   event to hook automatically; it can accept "", "clone", "fork", "add" or "never" (default "never")
```

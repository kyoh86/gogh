## gogh overlay add

Add an overlay file

```
gogh overlay add <source-path> <target-path> [flags]
```

### Options

```
      --for-init         Apply this overlay for 'gogh create' command
  -h, --help             help for add
  -p, --pattern string   Pattern to match repositories (e.g., 'github.com/owner/repo', '**/gogh')
```
